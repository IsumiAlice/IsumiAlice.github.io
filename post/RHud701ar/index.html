睡眠問題

睡眠後螢幕關閉，但滑鼠亮著，機箱風扇也在工作

解決方法：把登録編輯程式的 AwayModeEnable 設定為 0

位置在：電腦\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\Session Manager\Power

如果Windows無法安裝到指定磁盤

  1. shift+F10
  2. diskpart
  3. list disk
  4. select disk
  5. Clean
  6. 若需要 convert mbr
